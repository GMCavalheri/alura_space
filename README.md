# ALURA-SPACE

This project is a POC for the use of Django library from Python.

![Aplication view 1](/imgs/img1.png)


## Libraries Used

The main libraries used, specified in [`requirements.txt`](requirements.txt), are:

1. **[asgiref](https://pypi.org/project/asgiref/)** - standard for Python asynchronous web apps and servers.

2. **[Django](https://www.djangoproject.com/)** - high-level Python web framework 

3. **[python-dotenv](https://pypi.org/project/python-dotenv/)** - helps in the development of applications.

4. **[sqlparse](https://pypi.org/project/sqlparse/)** - non-validating SQL parser for Python.

5. **[tzdata](https://pypi.org/project/tzdata/)** - binaries for the IANA time zone database.


## Conclusion

It's possible to use Python Django library to create functional static pages.
